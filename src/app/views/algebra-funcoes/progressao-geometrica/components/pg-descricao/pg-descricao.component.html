Definição

Descrição

Progressão geométrica (P.G.) é uma sequência numérica em que cada termo, a partir do segundo, é obtido multiplicando-se o anterior por uma constante q, chamada razão da P.G.

Classificação

Uma progressão geométrica pode ser classificada da seguinte maneira:
•Crescente (a < 0: 0 < q < 1; a> 0: q > 1);
•Decrescente (a < 0: q> 1; a > 0: 0 < q < 1);
•Constante (a 0: q = 1; a = 0: q);
•Estacionária (q = 0);
•Oscilante (para qualquer a: q < 0)

Propriedades de uma Progressão Geométrica

I. Numa P.G. com número ímpar de elementos, temos que o quadrado do termo médio é igual ao produto dos extremos.
II. O produto dos termos equidistantes dos extremos de uma P.G. é igual ao produto desses extremos.
